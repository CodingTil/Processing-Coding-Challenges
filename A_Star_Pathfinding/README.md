## Usage
| Tool | Shortcut |
| --- | --- |
| Set starting point | `s` |
| Set end point | `e` |
| Simulate quickly | `f` |
| Simulate step by step | *space* |
| Add walls | *left-mouse* |
| Remove walls | *right-mouse* |
